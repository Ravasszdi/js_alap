# Gyakorló feladat

A feladatokat külön függvényekkel oldd meg. a függvényeket a fuggvenyek.js fájlban hozd létre és a főprogramból hívd meg.  

## 1. feladat

Érd el, hogy az input mezőbe beírt szám 1 és 5 közötti érték lehessen! 
Az OK gombra kattintva hívj meg egy függvényt, amiben írd ki az eredmeny azonosítójú elembe szövegesen az értékelést (1- elégtelen, 2- elégséges, 3- közepes, 4-jó, 5-jeles)

## 2. feladat

1. Írj függvényt, amiben generálj 12 véletlen számot a [-10, 100] intervallumban. A generált számokat tárold listában. A függvény visszatérési értéke a lista legyen. 

**Segítség:** 
- Véletlen szám: Math.random() [0,1) közötti számot generál
- Kerekítés: Math.floor() 

Ha nem tudod kitalálni, hoyg lesznek meg az egész számok, kérdezd meg az AI-t, de kérj tőle általános szabályt! Írd ide, mi a szabály [a,b] intervallumban lévő véletlen számok generálására. 

2. Írj függvényt, aminek bemeneti paramétere az előbbi függvéyn által visszaadott lista (a véletlen számokkal). A számokat írd ki divekbe az aside elembe. (annyi div legyen, ahány eleme van a listának. ) A diveket formázd css-sel, legyenek kör alakúak!

## 3. feladat

A adat.js fájlban találod az objektumlistát. Ezt már beimportáltam az index.js fájlba, tehát ott már használható. 
Írj függvényt, ami paraméterként kapja ezt a listát, és megjeleníti az adatokat egy táblázatban, az article elemben. Formázd is a táblázatot!
